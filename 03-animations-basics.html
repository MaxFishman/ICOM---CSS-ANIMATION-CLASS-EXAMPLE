CSS Animations Basics

CSS animations use @keyframes to create complex motions that run automatically, without user interaction. Unlike transitions, animations can loop, reverse, and have multiple intermediate steps.

Key Differences: Transitions vs Animations

Transitions

  • Require a trigger (hover, focus, etc.)
  • Simple: start state → end state
  • Perfect for interactions
  • Limited control over timing

Animations

  • Run automatically when applied
  • Complex: multiple keyframe steps
  • Great for attention-grabbing effects
  • Full control over timing and repetition

Basic Animation Structure

/* 1. Define keyframes */ @keyframes animationName { 0% { /* Starting state */ } 50% { /* Midpoint state */ } 100% { /* Ending state */ } } /* 2. Apply animation to element */ .element { animation: animationName 2s ease-in-out infinite; /* name duration timing iteration */ }

Fundamental Animation Types

Spin Animation

SPIN
@keyframes spin { from { transform: rotate(0deg); } to { transform: rotate(360deg); } } .spin { animation: spin 2s linear infinite; }

Pulse Animation

PULSE
@keyframes pulse { 0% { transform: scale(1); opacity: 1; } 50% { transform: scale(1.2); opacity: 0.7; } 100% { transform: scale(1); opacity: 1; } } .pulse { animation: pulse 1.5s ease-in-out infinite; }

Bounce Animation

BOUNCE
@keyframes bounce { 0%, 20%, 50%, 80%, 100% { transform: translateY(0); } 40% { transform: translateY(-20px); } 60% { transform: translateY(-10px); } } .bounce { animation: bounce 1s ease-in-out infinite; }

Fade In/Out

FADE
@keyframes fadeInOut { 0% { opacity: 0; } 50% { opacity: 1; } 100% { opacity: 0; } } .fade { animation: fadeInOut 2s ease-in-out infinite; }

Slide Animation

SLIDE
@keyframes slideLeftRight { 0% { transform: translateX(-50px); } 50% { transform: translateX(50px); } 100% { transform: translateX(-50px); } } .slide { animation: slideLeftRight 2s ease-in-out infinite; }

Color Change

COLOR
@keyframes colorChange { 0% { background-color: #2f6df6; } 25% { background-color: #22c55e; } 50% { background-color: #f59e0b; } 75% { background-color: #ef4444; } 100% { background-color: #2f6df6; } } .color { animation: colorChange 3s ease-in-out infinite; }

3. animation-direction

Controls the direction of animation playback:

normal
→
animation-direction: normal; /* Always left to right */
reverse
←
animation-direction: reverse; /* Always right to left */
alternate
↔
animation-direction: alternate; /* Forward, then backward */
alternate-reverse
↔
animation-direction: alternate-reverse; /* Backward, then forward */

4. animation-fill-mode

What styles apply before/after animation:

none
none
animation-fill-mode: none; /* No persistent styles */
forwards
→
animation-fill-mode: forwards; /* Keep final state */
backwards
←
animation-fill-mode: backwards; /* Apply initial state */
both
↔
animation-fill-mode: both; /* Initial + final states */
