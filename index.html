Tip: start with transition: transform 0.4s ease; and @keyframes spin { ... } to explore timing, delay, and iteration settings.

Transition properties

  • transition-property – which CSS property should animate
  • transition-duration – how long the change takes
  • transition-timing-function – easing curve
  • transition-delay – wait time before starting
  • Shorthand: transition: property duration timing delay;

Animation properties

  • @keyframes – define key steps of motion
  • animation-name and animation-duration
  • animation-iteration-count – repeats (e.g., infinite)
  • animation-direction and animation-fill-mode
  • Shorthand: animation: name duration timing delay count direction;
