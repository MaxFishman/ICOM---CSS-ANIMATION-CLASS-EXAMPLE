CSS Transitions Basics

CSS transitions allow you to smoothly animate changes between different states of an element. They're perfect for hover effects, focus states, and simple interactions.

What Are Transitions?

Transitions animate the change from one CSS property value to another. Instead of an instant change, you get a smooth animation over a specified duration.

Basic Transition Syntax
/* Basic syntax */
.element {
    transition: property duration timing-function delay;
}

/* Examples */
.box { transition: background-color 0.3s ease; }
.box { transition: all 0.5s ease-in-out; }
.box { transition: transform 0.2s cubic-bezier(0.25, 0.46, 0.45, 0.94); }

Interactive Examples

1. transition-property: Specific Property

This example only animates the background-color, other properties change instantly.

Hover me!
.property-demo {
    transition-property: background-color;
    transition-duration: 0.5s;
}

.property-demo:hover {
    background-color: #22c55e;
}
2. Quick Duration (0.1s)

Very fast transitions create snappy, responsive interactions.

Quick!
.duration-demo {
    transition: all 0.1s ease;
}

.duration-demo:hover {
    transform: scale(1.1);
    background-color: #f59e0b;
}
3. Custom Timing Function

Bounce effect using cubic-bezier timing function.

Bounce!
.timing-demo {
    transition: transform 0.8s cubic-bezier(0.68, -0.55, 0.265, 1.55);
}

.timing-demo:hover {
    transform: rotate(360deg) scale(1.2);
}
4. Transition Delay

Wait 0.5 seconds before starting the animation.

Wait for it...
.delay-demo {
    transition: all 0.3s ease;
    transition-delay: 0.5s;
}

.delay-demo:hover {
    transform: translateY(-20px);
    box-shadow: 0 10px 20px rgba(0,0,0,0.2);
}
5. Multiple Transitions

Different properties with different timing and delays.

Complex!
.multiple-demo {
    transition: 
        background-color 0.3s ease,
        transform 0.3s ease 0.1s,
        border-radius 0.3s ease 0.2s;
}

.multiple-demo:hover {
    background-color: #9333ea;
    transform: scale(1.1);
    border-radius: 50%;
}
