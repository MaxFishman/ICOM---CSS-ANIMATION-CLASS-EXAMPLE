# CSS Transitions & Animations - Complete Interactive Tutorial

A comprehensive, multi-page interactive tutorial for learning CSS transitions and animations from basics to advanced techniques.

## 🎯 What You'll Learn

This tutorial covers everything from simple hover effects to complex multi-element animations, with **interactive examples you can edit and experiment with** on each page.

## 📚 Tutorial Pages

### 1. [Transitions Basics](01-transitions-basics.html)

- Basic transition syntax and properties
- Live CSS editor with instant preview
- Hover effects and simple interactions
- Multiple transition examples

### 2. [Transition Properties Deep Dive](02-transition-properties.html)

- transition-duration comparison (0.1s to 2s)
- Timing functions (ease, linear, bounce, cubic-bezier)
- Property-specific transitions
- Interactive transition builder with sliders

### 3. [Animation Basics](03-animations-basics.html)

- @keyframes syntax and structure
- Animation vs Transition differences
- Duration, iteration, direction examples
- Custom animation builder tool

### 4. [Keyframes Examples](04-keyframes-examples.html)

- 2-step animations (0% → 100%)
- 3-step animations with midpoints
- 4+ step complex sequences
- Keyframe best practices and optimization

### 5. [Square Movement Demo](05-square-movement.html) ⭐ **Special Feature**

- Square inside square moving in clockwise pattern
- **Top-left → Top-right → Bottom-right → Bottom-left → Back to start**
- Multiple movement variations (counter-clockwise, bounce, smooth)
- Custom path editor with live preview
- Coordinate calculation explained

### 6. [Advanced Demos](06-advanced-demos.html)

- Particle systems and multi-element animations
- 3D transformations and perspective effects
- Performance optimization techniques
- Real-world animation patterns

## ✨ Interactive Features

- **Live Code Editors**: Edit CSS and see changes instantly
- **Interactive Controls**: Sliders, buttons, and form controls to modify animations
- **Custom Builders**: Create your own animations with guided tools
- **Mobile Friendly**: All examples work on mobile devices
- **Performance Tips**: Learn optimization techniques throughout
