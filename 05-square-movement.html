Square Movement Demo

Learn how to create smooth movement patterns by animating a square inside another square. This demonstrates positional animation and the clockwise movement pattern you requested.

Movement Pattern Breakdown

The square moves in a clockwise pattern: Top-Left → Top-Right → Bottom-Right → Bottom-Left → Back to Start

1
0% - 25%
Move from top-left corner to top-right corner
2
25% - 50%
Move from top-right corner to bottom-right corner
3
50% - 75%
Move from bottom-right corner to bottom-left corner
4
75% - 100%
Move from bottom-left corner back to top-left corner

Basic Clockwise Movement

Main Demo - Clockwise with Color Changes

Complete CSS Code

/* The outer container */ .outer-square { width: 300px; height: 300px; border: 3px solid #2f6df6; position: relative; background: #f8faff; } /* The moving square */ .inner-square { width: 60px; height: 60px; background: #2f6df6; position: absolute; animation: clockwiseSquare 4s ease-in-out infinite; } /* The movement keyframes */ @keyframes clockwiseSquare { 0% { top: 20px; left: 20px; background-color: #2f6df6; transform: rotate(0deg); } 25% { top: 20px; left: 220px; background-color: #22c55e; transform: rotate(90deg); } 50% { top: 220px; left: 220px; background-color: #f59e0b; transform: rotate(180deg); } 75% { top: 220px; left: 20px; background-color: #ef4444; transform: rotate(270deg); } 100% { top: 20px; left: 20px; background-color: #2f6df6; transform: rotate(360deg); } }

Movement Variations

Counter-Clockwise Movement

CCW
@keyframes counterClockwise { 0% { top: 20px; left: 20px; } 25% { top: 220px; left: 20px; } /* Down */ 50% { top: 220px; left: 220px; } /* Right */ 75% { top: 20px; left: 220px; } /* Up */ 100% { top: 20px; left: 20px; } /* Left */ }

Fast Bounce Movement

FAST
@keyframes fastBounceSquare { 0%, 100% { top: 20px; left: 20px; } 12.5%, 37.5%, 62.5%, 87.5% { transform: scale(1.2); /* Bounce at corners */ } 25% { top: 20px; left: 220px; } 50% { top: 220px; left: 220px; } 75% { top: 220px; left: 20px; } } .fast-bounce-demo { animation: fastBounceSquare 2s cubic-bezier(0.68, -0.55, 0.265, 1.55) infinite; }

Smooth Curve with Transform

SMOOTH
/* Using transform instead of top/left */ @keyframes smoothCurve { 0% { transform: translate(20px, 20px) rotate(0deg); border-radius: 8px; } 25% { transform: translate(220px, 20px) rotate(90deg); border-radius: 50%; /* Shape change */ } 50% { transform: translate(220px, 220px) rotate(180deg); border-radius: 8px; } 75% { transform: translate(20px, 220px) rotate(270deg); border-radius: 50%; } 100% { transform: translate(20px, 20px) rotate(360deg); border-radius: 8px; } }

Key Learning Points

1. Position vs Transform

  • top/left properties: Direct positioning, easier to understand coordinates
  • transform: translate(): Better performance, GPU-accelerated, smoother animation

2. Timing Functions Impact

  • ease-in-out: Smooth acceleration and deceleration at each corner
  • linear: Constant speed throughout the animation
  • cubic-bezier(): Custom timing for bounce effects

3. Coordinate Calculation

For a 300×300px container with 60×60px moving element and 20px padding:

  • Top-Left: (20px, 20px)
  • Top-Right: (220px, 20px) = (300 - 60 - 20, 20)
  • Bottom-Right: (220px, 220px) = (300 - 60 - 20, 300 - 60 - 20)
  • Bottom-Left: (20px, 220px) = (20, 300 - 60 - 20)
