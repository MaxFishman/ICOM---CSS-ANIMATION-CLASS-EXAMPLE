Keyframe Examples

Keyframes are the foundation of CSS animations. They define specific points in time during the animation sequence. Let's explore different patterns and complexities.

Understanding Keyframe Percentages

Keyframe Timing Breakdown:

  • 0% = Animation start (same as from)
  • 25% = 1/4 through the animation
  • 50% = Halfway point
  • 75% = 3/4 through the animation
  • 100% = Animation end (same as to)

Pro tip: You can use any percentage value (like 33%, 67%, 85%) for precise timing control!

Simple 2-Step Keyframes (0% → 100%)

Slide In
Move from left with fade-in effect
SLIDE
@keyframes slideIn { from { transform: translateX(-100px); opacity: 0; } to { transform: translateX(0); opacity: 1; } }
Grow & Shrink
Simple scale animation
GROW
@keyframes growShrink { 0% { transform: scale(1); } 100% { transform: scale(1.5); } }
Fade & Slide Up
Combine opacity and transform
FADE
@keyframes fadeSlide { 0% { opacity: 0; transform: translateY(20px); } 100% { opacity: 1; transform: translateY(0); } }

3-Step Keyframes (0% → 50% → 100%)

Bounce In
Overshoot effect with 3 keyframes
BOUNCE
@keyframes bounceIn { 0% { transform: scale(0); opacity: 0; } 50% { transform: scale(1.2); opacity: 0.8; } 100% { transform: scale(1); opacity: 1; } }
Wiggle
Back and forth rotation
WIGGLE
@keyframes wiggle { 0% { transform: rotate(0deg); } 50% { transform: rotate(10deg); } 100% { transform: rotate(-10deg); } }
Heartbeat
Rhythmic pulsing effect
❤️
@keyframes heartbeat { 0% { transform: scale(1); } 25% { transform: scale(1.1); } 50% { transform: scale(1); } 75% { transform: scale(1.05); } 100% { transform: scale(1); } }

4-Step Keyframes (0% → 25% → 50% → 75% → 100%)

Roll In
Complex entry animation with multiple phases
ROLL
@keyframes rollIn { 0% { transform: translateX(-100px) rotate(-180deg); opacity: 0; } 33% { transform: translateX(0) rotate(0deg); opacity: 1; } 66% { transform: translateX(20px) rotate(20deg); } 100% { transform: translateX(0) rotate(0deg); } }
Color Wave
Cycle through multiple colors
COLOR
@keyframes colorWave { 0% { background-color: #2f6df6; } 25% { background-color: #22c55e; } 50% { background-color: #f59e0b; } 75% { background-color: #ef4444; } 100% { background-color: #2f6df6; } }
Morph Shape
Transform between different shapes
MORPH
@keyframes morphShape { 0% { border-radius: 6px; background-color: #2f6df6; } 25% { border-radius: 25px; background-color: #22c55e; } 50% { border-radius: 50%; background-color: #f59e0b; } 75% { border-radius: 0; background-color: #ef4444; } 100% { border-radius: 6px; background-color: #2f6df6; } }

Complex Multi-Step Animations

Complex Path
Multiple transforms combined with color changes
PATH
@keyframes complexPath { 0% { transform: translate(0, 0) rotate(0deg); } 20% { transform: translate(50px, -30px) rotate(72deg); } 40% { transform: translate(80px, 20px) rotate(144deg); } 60% { transform: translate(30px, 50px) rotate(216deg); } 80% { transform: translate(-30px, 30px) rotate(288deg); } 100% { transform: translate(0, 0) rotate(360deg); } }
Elastic Effect
Fine-tuned elastic animation with 10 keyframes
ELASTIC
@keyframes elastic { 0% { transform: scale(1); } 10% { transform: scale(1.1, 0.9); } 20% { transform: scale(0.9, 1.1); } 30% { transform: scale(1.05, 0.95); } 40% { transform: scale(0.95, 1.05); } 50% { transform: scale(1.02, 0.98); } /* ... continues to 100% */ }

Keyframe Best Practices

1. Strategic Keyframe Placement

Fewer keyframes = smoother animation

  • Use 2-3 keyframes for simple effects
  • Use 4-6 keyframes for complex motions
  • Use 8+ keyframes only for very specific effects

2. Timing Distribution

Non-uniform timing creates natural motion

/* Good: Natural spacing */ @keyframes naturalBounce { 0% { transform: translateY(0); } 30% { transform: translateY(-60px); } /* Quick up */ 60% { transform: translateY(-20px); } /* Slow down */ 100% { transform: translateY(0); } /* Fall back */ } /* Avoid: Uniform spacing feels robotic */ @keyframes roboticBounce { 0% { transform: translateY(0); } 25% { transform: translateY(-20px); } 50% { transform: translateY(-40px); } 75% { transform: translateY(-20px); } 100% { transform: translateY(0); } }

3. Performance Optimization

Animate only transform and opacity for best performance

/* Fast: Uses GPU acceleration */ @keyframes performant { 0% { transform: translateX(0) scale(1); opacity: 1; } 100% { transform: translateX(100px) scale(1.2); opacity: 0.5; } } /* Slow: Causes layout recalculation */ @keyframes heavy { 0% { left: 0; width: 100px; margin: 10px; } 100% { left: 100px; width: 120px; margin: 20px; } }

4. Keyframe Naming

Use descriptive names that explain the motion

/* Good: Descriptive names */ @keyframes slideInFromLeft { /* ... */ } @keyframes pulseGently { /* ... */ } @keyframes bounceInElastic { /* ... */ } /* Poor: Generic names */ @keyframes anim1 { /* ... */ } @keyframes move { /* ... */ } @keyframes effect { /* ... */ }
